Para conectar con Google Sheets necesitas:
  1. Crear una Google Sheet y copiar su ID (el trozo largo de la URL)
  2. Obtener una API Key de Google Cloud Console (gratis)
  3. Pegar ambos valores aquí

📖 Guía de configuración
Paso 1: Crear la Google Sheet
  1. Ve a sheets.google.com y crea una hoja nueva
  2. Crea 4 pestañas: Recetas, Ingredientes, Stock, Menu
  3. Copia el ID de la URL (la parte entre /d/ y /edit)
Paso 2: Cabeceras necesarias
Recetas: id | nombre | tipo | reutilizable | tiempo | notas
Ingredientes: receta_id | ingrediente | cantidad | unidad
Stock: ingrediente | unidad | cantidad_disponible
Menu: dia | comida | cena
Paso 3: API Key gratuita
  1. Ve a console.cloud.google.com
  2. Crea un proyecto nuevo (o usa uno existente)
  3. Busca "Google Sheets API" y actívala
  4. Ve a Credenciales → Crear credencial → Clave de API
  5. Copia la clave generada
  6. Importante: en la Sheet ve a Compartir → "Cualquiera con el enlace puede ver"
💡 Con la API Key solo tienes acceso de lectura. Para escritura (guardar cambios) necesitarás OAuth2 o usar Apps Script. En modo offline, todos los datos se guardan localmente en tu navegador.
📖 Configurar Apps Script (escritura)
Paso 1: Abrir el editor de Apps Script
  1. Abre tu Google Sheet
  2. Menú superior → Extensiones → Apps Script
  3. Se abre un editor de código en una pestaña nueva
Paso 2: Pegar el código
  1. Borra todo el contenido del archivo Code.gs
  2. Pega el código del archivo apps-script.js que te he dado
  3. Pulsa el icono de 💾 Guardar (o Ctrl+S)
Paso 3: Publicar como aplicación web
  1. Pulsa el botón azul "Implementar" → "Nueva implementación"
  2. Tipo: Aplicación web
  3. Ejecutar como: Yo (tu cuenta)
  4. Quién tiene acceso: Cualquier usuario
  5. Pulsa Implementar
  6. Copia la URL de la aplicación web que aparece
La URL tendrá este aspecto:
https://script.google.com/macros/s/AKfycb.../exec
Paso 4: Pega esa URL en el campo "URL del Apps Script" de la configuración de la app.
⚠️ La primera vez que ejecutes una acción de escritura, Google te pedirá que autorices los permisos. Acepta con tu cuenta de Google.
